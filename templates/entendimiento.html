Entendimiento del Negocio y los Datos

Esta sección proporciona un contexto general del proyecto, alineado con los Objetivos de Desarrollo Sostenible (ODS).

Objetivo del Proyecto

El objetivo principal del proyecto es desarrollar un modelo de Machine Learning capaz de identificar y clasificar imágenes de flores en diferentes categorías (daisy, dandelion, rose, sunflower y tulip), este modelo busca automatizar el proceso de reconocimiento visual, reduciendo el error humano y permitiendo una clasificación rápida y precisa.

Contexto y Problemática

En el ámbito de la botánica y la agricultura, la clasificación de especies florales es una tarea fundamental para estudios de biodiversidad, control de cultivos y preservación de especies tradicionalmente, este proceso requiere de especialistas y tiempo considerable, el reto surge al intentar automatizar este proceso utilizando técnicas de aprendizaje automático

Evaluación del Procesamiento de Datos

Durante el procesamiento, se trabajó con un conjunto de datos compuesto por imágenes de flores distribuidas en cinco carpetas, cada una correspondiente a una clase, el modelo se entrena con una parte de las imágenes y se valida con el resto, evaluando su rendimiento mediante métricas como la precisión y el reporte de clasificación, se logró una precisión significativa, lo que demuestra la capacidad del modelo SVM para reconocer patrones visuales en imágenes previamente escaladas y estandarizadas.
