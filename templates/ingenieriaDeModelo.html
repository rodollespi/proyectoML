Ingeniería de modelo

Esta sección describe el proceso de preparación de datos, selección de algoritmos, entrenamiento del modelo SVM y el ajuste de hiperparámetros para clasificar diferentes tipos de flores a partir de imágenes.

Selección de Algoritmos

Para este proyecto se evaluaron algoritmos clásicos de clasificación supervisada. Finalmente, se seleccionó un Máquina de Vectores de Soporte (SVM) debido a:

- Su buen desempeño en problemas de alta dimensionalidad (como imágenes convertidas a vectores).

- Su capacidad para manejar datos linealmente separables mediante el uso de un kernel lineal.

- Su eficiencia con conjuntos de datos pequeños y medianos.

Entrenamiento del Modelo

Para entrenar el clasificador, se realizó el siguiente flujo de trabajo:

Carga de imágenes

-Las imágenes se organizaron por carpetas, donde cada carpeta representa una clase de flor.

Preprocesamiento de imágenes

- Redimensionamiento a 64×64 píxeles.

- Conversión a escala de grises.

- Aplanamiento del arreglo para transformarlo en un vector numérico.

Estandarización de características

- Se aplicó StandardScaler para centralizar y normalizar los datos, mejorando la eficiencia del SVM.

Codificación de etiquetas

- Se utilizó LabelEncoder para convertir los nombres de las flores a valores numéricos.

División de datos

El conjunto de datos se dividió en:

- 80% para entrenamiento

- 20% para prueba

Entrenamiento del modelo SVM

El clasificador SVM se entrenó utilizando los datos preprocesados. Después, se evaluó su rendimiento con accuracy_score y un classification_report.

Finalmente, el modelo, el escalador y el codificador se guardaron en archivos .pkl mediante joblib para su uso en despliegue.

Limpieza y Transformación

Se aplicaron procesos de normalización y escalado mediante StandardScaler para estandarizar los valores de los píxeles, mejorando la convergencia del modelo, las imágenes se convirtieron a escala de grises y se redimensionaron a 64x64 píxeles, reduciendo la complejidad computacional sin perder información relevante.
